5.2 Exemplo de Entrada no Glossário

Aqui está um exemplo de como uma entrada no glossário será documentada:

Termo: Dados Pessoais

Definição: Dados pessoais referem-se a qualquer informação relacionada a uma pessoa identificada ou identificável. Isso inclui, mas não se limita a, nomes, endereços, números de identificação, dados biométricos, informações de contato e qualquer outra informação que possa ser usada para identificar diretamente ou indiretamente uma pessoa física.

Sinônimos: Informações pessoais, dados de identificação pessoal (DIP)

Exemplos: Um exemplo de dados pessoais é o número de CPF (Cadastro de Pessoas Físicas) no Brasil, que é usado para identificar de forma única cada cidadão brasileiro.

Contexto: Dados pessoais são protegidos por regulamentações de privacidade, como o GDPR na União Europeia e a LGPD no Brasil. É importante garantir que o tratamento desses dados esteja em conformidade com as leis e regulamentos aplicáveis.

Responsável: Equipe de conformidade com a privacidade de dados.

Data de Revisão: 26 de março de 2024.
